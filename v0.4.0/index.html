FourierFlows.jl Documentation

Installation

Install FourierFlows.jl from the julia REPL prompt with

using Pkg
Pkg.add("FourierFlows")

Overview

FourierFlows provides a framework to write solvers for partial differential equations on periodic domains with Fourier-based pseudospectral methods. We support 1- and 2-dimensional domains at the moment.

Examples

For examples of FourierFlows.jl in action, see GeophysicalFlows.jl

Developers
